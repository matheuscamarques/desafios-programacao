# 🎲 Um RPG de Programação 🎮

### 💻 Missão 001 - Otimizando a performance de busca em larga escala 💻

📜 História:

Você é um(a) programador(a) muito conhecido(a) no mundo Elixir. Certo dia, foi requisitado(a) para um grande desafio: otimizar a performance de uma busca em larga escala em um banco de dados distribuído.

O banco de dados possui milhões de informações, sendo assim, é extremamente importante que a busca seja rápida e eficiente. Por isso, você deve fazer uso de técnicas avançadas de cache e algoritmos eficientes para a busca.

Sua missão é garantir que a busca de informações no banco de dados seja realizada da forma mais rápida possível, evitando a sobrecarga de requisitos. 

🎯 Desafio:

Criar um microserviço em Elixir com Phoenix que realize buscas em um banco de dados distribuído com alta performance utilizando algoritmos de cache.

Para ser considerado um sucesso, é necessário que o tempo de busca seja no máximo de 2 segundos.

Dados de entrada:
- Palavra-chave para busca 

Dados de saída: 
- Lista com informações relacionadas à palavra-chave encontradas no banco de dados.

Tecnologias que podem ser utilizadas:
- Elixir
- Phoenix
- Hadoop
- Redis
- Memcached
